# Linear Search

## Aim
To search an element in an array using linear search technique.

## Pseudocode
ALGO LinearSearch(arr[], n, key)
FOR i ← 0 TO n−1
IF arr[i] = key
RETURN i
END IF
END FOR
RETURN -1
END ALGO


## Variables Used
- **arr[]** — Input array  
- **n** — Number of elements  
- **key** — Element to be searched  

## Algorithm Explanation
The algorithm checks each element sequentially until the key is found
or the array ends.

## Time Complexity
**O(n)**

## Space Complexity
**O(1)**
